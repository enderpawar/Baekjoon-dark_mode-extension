# 백준 다크모드 크롬 확장 프로그램

백준 온라인 저지(acmicpc.net)를 다크모드로 사용할 수 있게 해주는 크롬 확장 프로그램입니다.

## 설치 방법

1. 크롬 브라우저에서 `chrome://extensions/` 를 엽니다.
2. 우측 상단의 "개발자 모드"를 활성화합니다.
3. "압축해제된 확장 프로그램을 로드합니다" 버튼을 클릭합니다.
4. 이 프로젝트 폴더를 선택합니다.

## 기능

- 백준 온라인 저지 사이트 전체에 다크 테마 적용
- 눈의 피로를 줄이는 어두운 배경색
- 코드 가독성을 위한 최적화된 색상 조합
- 모든 페이지에서 자동으로 적용

## 파일 구조

- `manifest.json` - 확장 프로그램 설정 파일
- `darkmode.css` - 다크모드 스타일시트
- `README.md` - 이 파일

## 아이콘 추가하기 (선택사항)

아이콘을 추가하려면 다음 크기의 PNG 이미지를 만들어 프로젝트 폴더에 추가하세요:
- `icon16.png` (16x16 픽셀)
- `icon48.png` (48x48 픽셀)
- `icon128.png` (128x128 픽셀)

아이콘이 없어도 확장 프로그램은 정상적으로 작동합니다.

## 커스터마이징

`darkmode.css` 파일을 수정하여 색상을 원하는 대로 조정할 수 있습니다.

주요 색상:
- 배경색: `#1e1e1e`
- 패널 배경색: `#252526`
- 텍스트 색상: `#d4d4d4`
- 링크 색상: `#569cd6`
- 테두리 색상: `#3e3e42`
